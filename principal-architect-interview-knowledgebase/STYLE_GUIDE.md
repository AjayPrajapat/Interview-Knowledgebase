# Documentation Style Guide

## Naming
- Category folders: `NN_Category_Name`
- Subcategories: `NN_Subcategory_Name`
- Topic files: `NNN_topic-slug.md`

## Content Standards
- Use heading order consistently.
- Explain tradeoffs explicitly.
- Use concise, production-oriented examples.
- Include failure modes and mitigation patterns.

## Topic Section Order
1. Definition
2. Internal Working
3. Trade-offs
4. Production Example
5. Failure Modes
6. Interview Questions
