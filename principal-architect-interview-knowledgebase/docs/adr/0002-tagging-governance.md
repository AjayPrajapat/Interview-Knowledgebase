# ADR 0002: Tagging Governance

- Status: accepted
- Date: 2026-03-03
- Deciders: Architecture Council

## Context
Uncontrolled tags create search noise and duplicate concepts.

## Decision
Enforce controlled tag vocabulary in `meta/tags.yml`.

## Alternatives
Unrestricted contributor-defined tags.

## Consequences
Higher consistency, better indexing, lower entropy.
