# ADR 0001: Taxonomy Standard

- Status: accepted
- Date: 2026-03-03
- Deciders: Architecture Council

## Context
Repository must scale beyond 1000 topics with predictable navigation.

## Decision
Adopt numbered category and subcategory folder standards with required index files.

## Alternatives
Flat tag-only organization.

## Consequences
Improved discoverability and governance; slight upfront content overhead.
