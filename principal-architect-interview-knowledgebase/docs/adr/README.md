# Architecture Decision Records (ADR)

Use ADRs to capture durable architectural and governance decisions.

## Naming Convention
`NNNN-short-title.md` (example: `0003-frontmatter-schema.md`)

## Required Sections
- Status
- Context
- Decision
- Alternatives
- Consequences
- References

## Lifecycle Status
`proposed`, `accepted`, `superseded`, `deprecated`
