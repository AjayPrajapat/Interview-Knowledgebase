# fullstack-interview-knowledgebase

A complete, expandable Markdown knowledgebase for fullstack interview preparation from beginner to architect level.

## Structure

- `00-interview-process` to `60-industry-trends`
- One interview-focused Markdown file per category
- Consistent template across files for easy extension

## Template Standard

Each question file follows:

- `Title`
- `Difficulty`
- `Role`
- `Stage`
- `Question`
- `What Interviewer Evaluates`
- `Key Concepts`
- `Explanation`
- `Solution Approach`
- `Example Answer`
- `Extra Notes`

## Naming Convention

All files use kebab-case.

Example: `scenario-slow-api.md`, `system-design-chat-app.md`, `security-xss.md`

## How To Extend

1. Add new Markdown files to the relevant category folder.
2. Keep naming in kebab-case.
3. Preserve the standard template so answers remain consistent.
4. Add advanced variants by role/stage (e.g., `architect`, `screening`, `behavioral`).
